# Recruiter Notes

Waitily is presented here as a public showcase, not as a full open-source production release.

## What this project demonstrates

- Full-stack product thinking
- Next.js App Router architecture
- Server Actions and Route Handlers
- Prisma/PostgreSQL data access
- Demand aggregation workflows
- Offer ranking and disclosure logic
- Admin intelligence/dashboard logic
- SEO and JSON-LD implementation
- Security-conscious public portfolio packaging

## Why the production repository is private

The private repository contains production data access patterns, database schema details, operational configuration and business logic that should not be exposed publicly.

## Best files to review

- `README.md` for product and architecture overview
- `docs/CASE_STUDY.md` for product context
- `docs/TECHNICAL_ARCHITECTURE.md` for implementation structure
- `docs/ROUTE_MAP.md` for routes and modules
- `components-demo/` for safe code samples
