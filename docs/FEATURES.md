# Features

## Public Demand Flow

- Purchase request form
- Product query normalization
- Reference price capture
- Reference URL validation
- Email validation
- Automatic demand pool creation or joining
- Default seven-day demand window
- No-account participation

## Demand Pool Experience

- Pool page by slug
- Demand count
- Category display
- Average reference price
- Window start and end dates
- Time remaining labels
- Share action
- Related requests
- Negotiated offers
- Fallback alternatives
- Affiliate/cashback disclosure copy

## Admin Intelligence

- Demand pool ranking
- Opportunity score
- Strategic labels
- Category insights
- Growth detection
- Contact-density signal
- Founder decision suggestions
- Offer performance scoring
- Monetization potential estimation
- Behavioral insights from views, clicks and shares

## Technical Features

- Next.js App Router
- Server Actions
- Route Handlers
- TypeScript utility modules
- Prisma PostgreSQL client
- Supabase client integration
- SEO metadata
- JSON-LD generation
- Responsive Tailwind UI
- Vercel deployment structure
