# Showcase Scope

This repository is intentionally not a full open-source release of Waitily.

## Included

- README and recruiter notes
- product overview
- screenshots/showcase previews
- high-level architecture
- route map
- data model overview
- selected reusable component demos
- selected safe TypeScript helper examples

## Excluded

- production source repository
- environment variables
- secrets and credentials
- database schema dumps
- Prisma migrations
- real user data
- private operational logic
- production infrastructure configuration

## Reason

The purpose is to demonstrate full-stack product development ability while protecting the commercial project and its users.
